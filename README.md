## 📌 Sobre o Projeto

Este é o meu portfólio pessoal, desenvolvido para mostrar minha evolução como estudante de **Redes de Computadores** e meus projetos práticos. O site foi criado com foco em:

- ✅ Design limpo e profissional (inspirado no GitHub Dark Theme)
- ✅ Layout responsivo (funciona em celular, tablet e desktop)
- ✅ Apresentação clara de tecnologias e habilidades
- ✅ Links diretos para meus projetos no GitHub
